[プロヴァンス](https://ja.wikipedia.org/wiki/プロヴァンス)

南フランス南東部、地中海沿いの地域圏 (南は地中海に面し、東部はアルプス山脈をはさんでイタリアと国境を接する)
気候は地中海性気候の影響で一年を通して温暖で乾燥しており、世界中の著名人がニース、カンヌなどに別荘を構えることで有名
